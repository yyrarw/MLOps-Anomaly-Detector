MLOps Anomaly Detector

Это сервис, который автоматически находит аномалии в метриках ML-инфраструктуры.
ML-инженер разворачивает инференс-серверы (vLLM, Sglang и т.д.), которые обслуживают запросы к моделям. Эти серверы генерируют кучу метрик — загрузка GPU, скорость генерации токенов, задержки, очереди запросов, потребление памяти. Этот сервис нужен для того чтобы следить за этим.
Пользователь экспортирует метрики в CSV и загружает в сервис. Система валидирует данные, а валидные прогоняет через одну из моделей детекции на выбор. Каждая модель стоит разное количество кредитов. На выходе — список найденных аномалий с указанием времени, серьёзности и возможной причины.
